Pre-Kindergarten

Pre-K Math Interventions

Number and Operations

  • **Counting Objects 1-10 with One-to-One Correspondence:** Students use manipulatives (e.g., counting bears, unifix cubes) to count a given set, physically touching each object as they say the number. Practice counting up to 10 objects.

    Assessment: Direct observation, count objects to 10 on command, informal checklist.

  • **Numeral Identification 0-5:** Match numerals to quantities using flashcards, number puzzles, or sensory bins. Trace numerals in sand or shaving cream.

    Assessment: Point to requested numeral, match numeral to set of objects (0-5), flashcard recognition.

  • **Quantity Comparison (More, Less, Same):** Use visual aids and concrete objects to compare sets. Engage in games like "Which pile has more?" or "Show me less than 3."

    Assessment: Student correctly identifies "more," "less," or "same" when comparing two small sets (up to 5).

  • **Sorting by Attributes (Color, Size, Shape):** Provide diverse sets of objects (blocks, buttons) and guide students to sort by color, size, shape, or type. Discuss the categories used.

    Assessment: Student sorts a given set of objects into correct categories based on a specified attribute.

  • **Subitizing Small Quantities (1-3):** Quickly recognize the number of objects in a small set without counting (e.g., dot patterns on dice, fingers).

    Assessment: Flash dot cards (1-3) for 1-2 seconds; student states the quantity immediately.

  • **Number Order (1-5):** Arrange number cards or objects in sequential order from 1 to 5.

    Assessment: Student correctly orders number cards 1-5 from a mixed set.

  • **Composing and Decomposing Numbers (up to 5):** Use part-part-whole mats and counters to show different ways to make a number (e.g., 5 can be 2 and 3).

    Assessment: Given a total (e.g., 4), student shows two different combinations using manipulatives.

  • **Identifying Ordinal Numbers (First, Second, Third):** Use a line of students or objects to practice identifying "first," "second," and "third."

    Assessment: Student points to the "first," "second," or "third" object in a small sequence.

  • **Simple Pattern Recognition (AB, AAB):** Create and extend simple patterns using blocks, beads, or movements.

    Assessment: Student identifies the repeating unit in a pattern and extends it by at least two elements.

  • **Number Writing Practice (0-5):** Practice forming numerals 0-5 using playdoh, tracing, or writing on whiteboards.

    Assessment: Student correctly writes numerals 0-5 when prompted (legibility check).

Geometry

  • **Basic Shape Recognition (Circle, Square, Triangle, Rectangle):** Use shape sorters, puzzles, and drawing activities for these core shapes. Go on a "shape hunt" in the classroom.

    Assessment: Student points to or names requested shapes, identifies shapes in real-world objects.

  • **Shape Attributes:** Describe attributes of shapes (e.g., "a square has four equal sides," "a circle is round").

    Assessment: Given a shape, student describes at least one defining attribute.

  • **Building with Shapes:** Use pattern blocks or tangrams to create new shapes or pictures, discussing the component shapes.

    Assessment: Student successfully combines 2-3 basic shapes to create a larger, identifiable shape.

  • **Identifying 3D Shapes (Sphere, Cube, Cylinder):** Introduce and identify real-world examples of spheres, cubes, and cylinders.

    Assessment: Student points to or names requested 3D shapes from a collection of objects.

  • **Spatial Relationships (In, On, Under):** Use positional words to describe the location of objects (e.g., "Put the block under the table").

    Assessment: Student correctly places an object based on verbal directional cues.

  • **Matching Shapes:** Play matching games with identical 2D shapes.

    Assessment: Student matches 5/5 identical shape pairs.

  • **Drawing Shapes:** Provide templates or verbal instructions for drawing basic shapes.

    Assessment: Student independently draws recognizable circles, squares, and triangles.

  • **Shape Puzzles:** Work on simple 2-4 piece shape puzzles to develop spatial reasoning.

    Assessment: Student completes a 3-piece shape puzzle independently.

  • **Sorting by Shape:** Provide a mixed set of objects and ask students to sort them by their geometric shape.

    Assessment: Student correctly sorts objects into bins labeled with different shapes.

  • **Creating Shape Pictures:** Encourage students to create pictures using various cut-out geometric shapes.

    Assessment: Student uses at least three different shapes to create a picture.

Measurement and Data

  • **Comparing Lengths (Longer/Shorter):** Use non-standard units (e.g., paperclips, blocks) to compare the length of two objects. Introduce terms like "longer," "shorter," "same length."

    Assessment: Given two objects, student correctly identifies which is "longer" or "shorter."

  • **Comparing Heights (Taller/Shorter):** Compare the height of students or objects using terms "taller" and "shorter."

    Assessment: Student correctly identifies the "taller" or "shorter" of two individuals/objects.

  • **Comparing Weights (Heavier/Lighter):** Use a balance scale or direct comparison to identify "heavier" and "lighter" objects.

    Assessment: Student predicts and then confirms which of two objects is "heavier" using a balance scale.

  • **Comparing Capacities (More/Less):** Use different sized containers and water/sand to explore "holds more" or "holds less."

    Assessment: Student identifies which container "holds more" when comparing two visibly different containers.

  • **Creating Simple Patterns (AB, AAB):** Students use colored blocks, beads, or actions to create and extend simple repeating patterns.

    Assessment: Student accurately creates and extends an AB pattern for at least three repetitions.

  • **Sorting and Classifying Objects:** Sort a collection of varied objects into categories based on one attribute (e.g., "all the red toys," "all the soft items").

    Assessment: Student sorts a set of 5-7 objects into two categories based on a given attribute.

  • **Non-Standard Measurement:** Measure objects using unifix cubes or blocks placed end-to-end.

    Assessment: Student measures a short object (e.g., a crayon) using 3-5 unifix cubes and states the number of cubes.

  • **Making Simple Bar Graphs (Pictographs):** Create a simple pictograph using sticky notes or drawings to represent data from a class poll (e.g., "Favorite Color").

    Assessment: Student contributes their data point to a class pictograph and interprets one simple question ("Which is most popular?").

  • **Ordering by Size:** Arrange 3-4 objects from smallest to largest or shortest to longest.

    Assessment: Student correctly orders 3 objects by length from shortest to longest.

  • **Understanding Time (Day/Night, Seasons):** Discuss daily routines associated with day and night, and talk about observable changes during different seasons.

    Assessment: Student identifies at least two activities associated with "day" and "night."

Pre-K Reading Interventions

Phonological Awareness

  • **Rhyming Games (Identification & Production):** Play "Does it rhyme?" with picture cards (e.g., cat/hat, dog/frog). Provide a word and ask for a rhyming word. Use nursery rhymes and rhyming books.

    Assessment: Student identifies rhyming words from a pair, or produces a rhyming word for a given CVC word.

  • **Syllable Clapping/Counting:** Clap out syllables in students' names, common objects, and animals. Segment words into syllables (e.g., 'el-e-phant').

    Assessment: Student accurately claps and counts syllables in 3-syllable words.

  • **Initial Sound Identification:** Focus on initial sounds in words using picture sorts (e.g., "Which pictures start with /b/ like 'ball'?"). Use initial sound isolation activities.

    Assessment: Student identifies the initial sound in CVC words (e.g., /m/ in 'mat').

  • **Sentence Segmentation:** Say a sentence and have the student clap for each word in the sentence.

    Assessment: Student claps and counts words in a simple 3-4 word sentence.

  • **Alliteration Games:** Play games focusing on words that start with the same sound (e.g., "Silly Susie Sings").

    Assessment: Student identifies words that start with the same sound when given a group of words.

  • **Compound Word Blending & Segmentation:** Practice putting two words together to make a compound word (e.g., 'sun' + 'flower' = 'sunflower') and breaking them apart.

    Assessment: Student correctly blends two words to form a compound word, and segments a compound word into its two parts.

  • **Oral Blending (Onset-Rime):** Provide onset and rime (e.g., /b/ + -at) and have students blend to form the word "bat."

    Assessment: Student blends 3/4 onset-rime pairs into a word.

  • **Oral Segmentation (Onset-Rime):** Give a word and have students break it into its onset and rime (e.g., "cat" -> /c/ + -at).

    Assessment: Student segments 3/4 given CVC words into onset and rime.

  • **Word Awareness:** Identify words within spoken sentences by pointing to discrete word cards as a sentence is read.

    Assessment: Student points to each word card as a 3-4 word sentence is read aloud.

  • **Auditory Discrimination of Sounds:** Listen to pairs of words and identify if they start with the same sound or different sounds.

    Assessment: Given pairs of words, student indicates "same" or "different" for initial sounds.

Concepts of Print

  • **Book Handling & Orientation:** Model and practice holding a book right side up, turning pages one by one, and identifying the front/back cover and title.

    Assessment: Student correctly holds a book, opens to the beginning, and turns pages sequentially.

  • **Print Awareness (Words vs. Letters):** Point to words as you read aloud. Explain that words tell the story and pictures help. Differentiate between letters, words, and sentences.

    Assessment: Student identifies a single word versus a single letter when pointed to in a book.

  • **Directionality (Left-to-Right, Top-to-Bottom):** Trace left-to-right reading direction with a finger. Emphasize returning to the left at the end of each line and moving top-to-bottom.

    Assessment: Student accurately tracks print with a finger as a sentence is read aloud.

  • **Return Sweep:** Practice moving eyes/finger from the end of one line to the beginning of the next.

    Assessment: Observer notes student's eye movement/finger tracking accuracy during reading aloud.

  • **Identifying Parts of a Book:** Explicitly teach and identify the author, illustrator, title, and spine of a book.

    Assessment: Student points to the author and title on a book cover when prompted.

  • **Punctuation Awareness:** Introduce and identify periods, question marks, and exclamation points. Explain their function in reading.

    Assessment: Student points to a period and explains it means "stop" or "end of a thought."

  • **Matching Spoken Word to Print:** Use large print text and point to each word as it is spoken, having students match spoken words to printed words.

    Assessment: Student accurately points to each word as a sentence is read aloud by the teacher.

  • **Tracking Print:** Use a pointer or finger to track the words as a shared reading text is read aloud.

    Assessment: Teacher observation of accurate print tracking during shared reading.

  • **First and Last Word in a Sentence:** Identify the first and last word of a sentence in large print.

    Assessment: Student correctly identifies the first and last word in a simple written sentence.

  • **Recognizing Repeated Words:** In a familiar text, identify and point to common repeated words (e.g., "the," "a," "is").

    Assessment: Student can locate and point to a specific high-frequency word repeated in a simple text.

Alphabet Knowledge

  • **Letter Recognition (Uppercase/Lowercase):** Use alphabet puzzles, letter matching games, and tactile letters (e.g., sandpaper letters, playdoh letters). Focus on 2-3 new letters per week, then review.

    Assessment: Rapid letter naming fluency (e.g., 26 letters in 30 seconds), matching uppercase to lowercase.

  • **Letter Sounds (Consonants & Short Vowels):** Introduce 2-3 letter sounds per week with corresponding pictures, actions, and songs. Focus on continuous sounds first (e.g., /m/, /s/).

    Assessment: Rapid letter sound fluency (e.g., 26 sounds in 30 seconds), saying sound for given letter.

  • **Letter Formation (Tracing & Writing):** Practice forming letters using multi-sensory methods like tracing in sand, shaving cream, or on whiteboards. Introduce correct letter formation strokes.

    Assessment: Student writes 5-10 target letters correctly with proper formation (observation, rubric).

  • **Alphabet Sequencing:** Arrange letter cards in alphabetical order. Sing the alphabet song while pointing to letters.

    Assessment: Student correctly orders 5-7 letter cards in alphabetical sequence.

  • **Matching Letters to Objects:** Match an alphabet letter to a picture that starts with that letter's sound.

    Assessment: Student matches 4/5 target letters to corresponding initial sound pictures.

  • **Identifying Letters in Own Name:** Help students recognize and spell the letters in their own name and family members' names.

    Assessment: Student correctly identifies all letters in their first name.

  • **Letter Hunts:** Go on letter hunts in books, environmental print, or a letter chart to find specific letters.

    Assessment: Student locates and circles 5 instances of a target letter in a short text.

  • **Capital vs. Lowercase Sorting:** Sort a mixed set of uppercase and lowercase letter cards into two categories.

    Assessment: Student accurately sorts 10 mixed letter cards into uppercase/lowercase categories.

  • **Writing First Letter of Name:** Practice writing the first letter of their name accurately.

    Assessment: Student writes the first letter of their name with correct formation.

  • **Letter Games (Bingo, Memory):** Play interactive letter games to reinforce recognition and sounds in a fun way.

    Assessment: Student correctly identifies letters/sounds during game play (e.g., wins a Bingo round by identifying letters).

Kindergarten

Kindergarten Math Interventions

Operations and Algebraic Thinking

  • **Composing and Decomposing Numbers to 10 (using Ten-Frames):** Use ten-frames and two-color counters to visually represent and explore different ways to make a number (e.g., 7 can be 5 and 2, or 3 and 4).

    Assessment: Student correctly shows 2-3 different ways to compose a target number (e.g., 6) using counters on a ten-frame.

  • **Addition within 5 (Concrete Models):** Use finger counting, unifix cubes, or drawings to model simple addition stories (e.g., "2 apples + 3 apples = ?"). Focus on conceptual understanding.

    Assessment: Student solves 3/5 addition problems within 5 using manipulatives and states the sum.

  • **Subtraction within 5 (Take Away):** Use concrete objects to act out "take away" subtraction problems (e.g., "5 cookies, eat 2, how many left?").

    Assessment: Student solves 3/5 subtraction problems within 5 using concrete objects and states the difference.

  • **"Add To" and "Take From" Word Problems within 10:** Solve simple word problems using pictures or concrete objects to represent the quantities. Focus on understanding the action.

    Assessment: Student accurately models and solves 2/3 simple addition/subtraction word problems within 10.

  • **Making 10 (Number Bonds):** Focus on pairs of numbers that add up to 10 (e.g., 6 and 4, 8 and 2). Use ten-frames or number bonds to visualize.

    Assessment: Given a number (e.g., 7), student states the missing number to make 10.

  • **Even and Odd Numbers (Introduction):** Sort groups of objects into pairs to determine if the number is even (all paired) or odd (one left over).

    Assessment: Student correctly sorts groups of 6-10 objects into even/odd categories.

  • **Representing Addition/Subtraction with Equations:** Write simple equations (e.g., $2+3=5$) after modeling or solving a problem.

    Assessment: Student writes a correct addition or subtraction equation for a given simple word problem.

  • **Comparing Number of Objects (Greater Than, Less Than):** Compare the number of objects in two groups (up to 10) using one-to-one correspondence and phrases like "greater than," "less than," or "equal to."

    Assessment: Student identifies the group with "greater than" or "less than" objects from two given groups.

  • **Finding the Missing Part (Part-Part-Whole):** Given a total and one part, use manipulatives to find the missing part (e.g., Total 8, Part 3, What's the other part?).

    Assessment: Student identifies the missing part in a part-part-whole model (sum up to 10).

  • **Counting On (from a given number):** Start counting from a number other than 1 (e.g., "Start at 4, count on to 7"). Use a number line or track.

    Assessment: Student accurately counts on from a given number to a specified target number (e.g., from 5 to 10).

Numbers and Operations in Base Ten

  • **Counting to 100 by Ones and Tens:** Practice rote counting and counting by tens (10, 20, 30...). Use a 100 chart to identify number patterns.

    Assessment: Student counts to 100 by ones, counts to 100 by tens (starting at 10).

  • **Number Identification 1-20:** Practice identifying numbers out of sequence. Play "number scavenger hunt" or "number bingo."

    Assessment: Student identifies 15/20 numerals when presented randomly.

  • **Writing Numerals 0-20:** Practice forming numerals 0-20 using playdoh, tracing, writing on whiteboards, and paper. Focus on correct stroke order.

    Assessment: Student writes 10 specified numerals (0-20) legibly and with correct formation.

  • **Counting Forward and Backward:** Practice counting forward from any given number up to 20, and backward from 10.

    Assessment: Student counts forward from 7 to 15, and backward from 10 to 0.

  • **Understanding Place Value (Tens and Ones Introduction):** Group 10 objects together to form a "ten." Count bundles of tens and individual ones (e.g., 2 tens and 3 ones make 23).

    Assessment: Given 10-20 objects, student groups them into tens and ones and states the total.

  • **Comparing Numbers to 100:** Use comparison language ("greater than," "less than," "equal to") with two numbers up to 100. Use a number line or 100 chart for visual support.

    Assessment: Student accurately compares two given numbers (e.g., 45 vs. 54) using appropriate language.

  • **Ordering Numbers to 20:** Arrange a set of 3-5 number cards (0-20) in sequential order.

    Assessment: Student correctly orders 5 shuffled number cards (e.g., 7, 12, 3, 18, 9).

  • **Identifying Numbers Before/After:** Given a number (e.g., 15), student states the number that comes immediately before or after.

    Assessment: Student correctly states the number before/after for 5/6 given numbers (up to 20).

  • **Number Representations (Tally Marks, Fingers, Objects):** Practice showing numbers in multiple ways (e.g., show 5 using fingers, tally marks, drawing dots).

    Assessment: Given a number (e.g., 7), student represents it using two different methods.

  • **Skip Counting by 5s to 50:** Practice skip counting by 5s. Relate it to counting nickels.

    Assessment: Student accurately skip counts by 5s to 50.

Measurement and Data

  • **Direct Comparison of Length, Weight, and Capacity:** Compare objects directly to determine which is "longer/shorter," "heavier/lighter," or "holds more/less."

    Assessment: Given two objects, student correctly compares them using the target comparative language.

  • **Classifying Objects and Counting Categories:** Sort objects into categories (e.g., by color, shape, size) and count the number of objects in each category.

    Assessment: Student sorts a set of mixed objects into at least three categories and accurately counts items in each.

  • **Using Non-Standard Units to Measure Length:** Measure short objects using non-standard units (e.g., paper clips, unifix cubes) placed end-to-end.

    Assessment: Student accurately measures a crayon or book using 5-10 unifix cubes and states the measurement.

  • **Creating Simple Picture Graphs:** Collect data from a class question (e.g., "Favorite Pet") and create a simple picture graph using drawings or stickers.

    Assessment: Student contributes data to a class graph and answers a simple question like "Which pet has the most votes?".

  • **Telling Time to the Hour:** Practice identifying the hour hand and minute hand, and telling time to the hour on an analog clock.

    Assessment: Student states the correct hour for 3-4 analog clock faces showing time to the hour.

  • **Identifying Coins (Penny, Nickel, Dime):** Recognize and name pennies, nickels, and dimes. State their values.

    Assessment: Student correctly identifies 3/4 common coins (penny, nickel, dime) and states their value.

  • **Comparing Temperatures (Hot/Cold):** Discuss differences between hot and cold, and identify appropriate clothing for different temperatures.

    Assessment: Student correctly identifies whether a picture depicts a "hot" or "cold" setting.

  • **Sequencing Events (First, Next, Last):** Order events in a story or daily routine using sequence words.

    Assessment: Student correctly sequences 3-4 picture cards depicting a routine (e.g., getting ready for school).

  • **Understanding Simple Patterns (AB, ABC):** Students identify the core unit of a pattern and extend it.

    Assessment: Student extends an ABC pattern using blocks for at least one full cycle.

  • **Estimating Length:** Practice estimating the length of common classroom objects before measuring with non-standard units.

    Assessment: Student makes a reasonable estimation (e.g., "about 5 cubes long") for a short object before measuring.

Kindergarten Reading Interventions

Phonological Awareness

  • **Phoneme Blending (CVC words):** Give individual sounds (e.g., /c/-/a/-/t/) and have students blend them to say the word "cat." Use Elkonin boxes with counters to visually represent sounds.

    Assessment: Student orally blends 8/10 CVC sounds into a word.

  • **Phoneme Segmentation (CVC words):** Have students break a word into individual sounds (e.g., "dog" -> /d/-/o/-/g/). Use Elkonin boxes or sound boxes.

    Assessment: Student orally segments 8/10 CVC words into individual phonemes.

  • **Onset-Rime Blending:** Provide onset and rime (e.g., /b/ + -at) and have students blend to form the word "bat." Practice with common rimes (-at, -ap, -en, -it, -og).

    Assessment: Student blends 8/10 onset-rime pairs into a word.

  • **Onset-Rime Segmentation:** Give a word and have students break it into its onset and rime (e.g., "fan" -> /f/ + -an).

    Assessment: Student segments 8/10 given CVC words into onset and rime.

  • **Phoneme Addition/Deletion:** Practice adding a sound to a word (e.g., "Add /s/ to 'top' -> 'stop'") or deleting a sound (e.g., "Say 'play' without /p/ -> 'lay'").

    Assessment: Student correctly performs 3/5 phoneme addition/deletion tasks.

  • **Discriminating Between Initial, Medial, and Final Sounds:** Identify where a specific sound is heard in a word (beginning, middle, or end).

    Assessment: Student correctly identifies the position (initial, medial, final) of a given sound in a CVC word.

  • **Auditory Sorting by Initial Sound:** Sort picture cards into groups based on their beginning sounds.

    Assessment: Student sorts 8/10 picture cards into correct initial sound categories.

  • **Rhyme Production:** Given a word, student generates one or more words that rhyme.

    Assessment: Student produces at least two rhyming words for 3/5 target words.

  • **Syllable Manipulation:** Remove a syllable from a compound word (e.g., "Say 'cupcake' without 'cup' -> 'cake'").

    Assessment: Student successfully removes a syllable from 3/4 compound words.

  • **Sentence Dictation with Word Cards:** Students arrange word cards to form a spoken sentence.

    Assessment: Student correctly arranges word cards to form a simple 3-5 word sentence.

Phonics

  • **Letter-Sound Correspondence (All Letters):** Use a multi-sensory approach (see, hear, touch) to teach letter sounds. Practice writing letters while saying their sounds. Use alphabet charts and tactile letters.

    Assessment: Student accurately states the sound for 26/26 uppercase and lowercase letters (letter sound fluency probe).

  • **CVC Word Reading:** Practice reading short a, e, i, o, u CVC words using decodable readers, word families, and magnetic letters. Focus on blending sounds smoothly.

    Assessment: Student accurately reads 15/20 CVC words from a list, or 90% accuracy in a decodable text.

  • **High-Frequency Words (Sight Words):** Introduce 3-5 new high-frequency words per week (e.g., "the," "and," "is," "a") using flashcards, word walls, and sentence building. Aim for instant recognition.

    Assessment: Student reads 25/30 kindergarten-level high-frequency words instantly (Dolch/Fry list).

  • **Consonant Digraphs (sh, ch, th, wh):** Explicitly teach the sounds of common consonant digraphs. Practice reading and sorting words containing these patterns.

    Assessment: Student correctly identifies the sound of each digraph and reads 8/10 words containing them.

  • **Blends (Initial L, R, S blends):** Introduce initial blends (e.g., bl, gr, st). Practice blending the two sounds smoothly.

    Assessment: Student reads 8/10 words with initial blends (e.g., 'slip', 'grab').

  • **Decoding Nonsense Words:** Practice reading nonsense CVC words to ensure students are applying phonetic rules rather than memorization.

    Assessment: Student reads 5/7 nonsense CVC words with correct sounds.

  • **Picture-Word Matching:** Match CVC words to corresponding pictures.

    Assessment: Student accurately matches 8/10 CVC words to pictures.

  • **Word Family Sorts:** Sort words by their word family (e.g., -at: cat, hat, bat; -en: hen, pen, ten).

    Assessment: Student correctly sorts 10 words into 2-3 word families.

  • **Dictation of CVC Words:** Say a CVC word; student writes the word using their letter-sound knowledge.

    Assessment: Student correctly spells 5/7 dictated CVC words.

  • **Sentence Reading (Simple Decodable):** Read simple sentences composed primarily of CVC words and high-frequency words.

    Assessment: Student reads 3/5 simple decodable sentences with 90% accuracy and fluency.

Reading Comprehension

  • **Retelling Stories (Beginning, Middle, End):** After reading a simple story aloud, ask students to retell the main events in order. Use sequence cards or a simple graphic organizer.

    Assessment: Student retells a familiar story including at least one event from the beginning, middle, and end.

  • **Answering Wh-Questions (Who, What, Where):** Ask "who," "what," and "where" questions about read-alouds or short decodable texts.

    Assessment: Student answers 3/4 'wh' questions accurately after listening to a short story.

  • **Making Connections (Text-to-Self):** Encourage students to connect story events or characters to their own experiences.

    Assessment: Student makes a relevant text-to-self connection to a read-aloud story.

  • **Identifying Main Character & Setting:** After reading, ask students to name the main character(s) and where the story takes place.

    Assessment: Student correctly identifies the main character and setting from a given simple story.

  • **Predicting:** Before and during reading, pause and ask students to predict what might happen next.

    Assessment: Student makes a reasonable prediction based on story clues.

  • **Visualizing:** After reading a descriptive passage, ask students to close their eyes and describe what they "saw" in their minds.

    Assessment: Student describes 2-3 visual details from a short descriptive passage.

  • **Understanding Simple Cause & Effect:** In a simple story, identify a cause and its effect (e.g., "The boy dropped the ball, so it rolled away.").

    Assessment: Student identifies the cause or effect in a simple picture sequence or sentence.

  • **Drawing to Show Comprehension:** Have students draw a picture that represents a key event or the main idea of a story.

    Assessment: Student's drawing accurately reflects a key detail or the main event of the story.

  • **Picture-Supported Questions:** Use picture books or decodable texts with visual support to ask comprehension questions.

    Assessment: Student answers 4/5 picture-supported comprehension questions correctly.

  • **Retelling with Props:** Provide props or puppets for students to use as they retell a story, helping them remember key events and characters.

    Assessment: Student uses props to sequence and retell a story with 80% accuracy of events.

1st Grade

1st Grade Math Interventions

Operations and Algebraic Thinking

  • **Fluency within 20 (Addition & Subtraction):** Daily timed practice with addition and subtraction facts using strategies like counting on, counting back, making ten, doubles, and near doubles. Use flashcards, fact triangles, and online games.

    Assessment: Timed fluency probe (e.g., 20 facts in 1 minute, aiming for 15+ correct).

  • **Understanding Relationship between Addition & Subtraction:** Use fact families (e.g., $3+4=7$, $4+3=7$, $7-3=4$, $7-4=3$) to demonstrate inverse operations.

    Assessment: Given three numbers (e.g., 5, 8, 13), student writes all four related equations.

  • **Solving Various Word Problem Types within 20:** Focus on "result unknown," "change unknown," and "start unknown" problems. Use concrete models, drawings, and equations to represent and solve problems.

    Assessment: Student accurately solves 3/4 different types of word problems within 20.

  • **Adding Three Numbers:** Use number lines or mental math strategies (e.g., making a 10 first) to add three single-digit numbers.

    Assessment: Student accurately solves 4/5 problems adding three single-digit numbers.

  • **Using the Equal Sign:** Emphasize that the equal sign means "is the same as" rather than "gives the answer." Solve problems like $7 = 5 + 2$.

    Assessment: Student correctly completes open equations like $6 + 3 = ? + 5$.

  • **Modeling with Part-Part-Whole:** Use part-part-whole mats to model addition and subtraction problems, focusing on finding the unknown part or whole.

    Assessment: Student correctly uses a part-part-whole mat to solve a given addition/subtraction problem.

  • **Comparing Equations:** Compare two addition or subtraction expressions using $<, >, =$ (e.g., $5+3 \text{ _ } 4+4$).

    Assessment: Student accurately places the correct comparison symbol for 3/4 equation pairs.

  • **Number Line Strategies for +/-:** Use an open number line to count on or count back for addition and subtraction within 20.

    Assessment: Student accurately uses an open number line to solve 3/4 addition or subtraction problems within 20.

  • **Fact Families to 10:** Reinforce mastery of all addition and subtraction facts within 10 through daily practice.

    Assessment: Student can orally state all four facts in a fact family for a given set of numbers (e.g., 2, 3, 5).

  • **Understanding True/False Equations:** Determine if addition/subtraction equations are true or false.

    Assessment: Student correctly identifies 4/5 true/false equations.

Numbers and Operations in Base Ten

  • **Understanding Place Value (Tens and Ones):** Use base-ten blocks (longs and units) to model numbers, identify the value of digits, and understand that 10 ones make a ten.

    Assessment: Given a two-digit number, student builds it with base-ten blocks and identifies the number of tens and ones.

  • **Adding Two-Digit Numbers (No Regrouping):** Use base-ten blocks or drawings to add two-digit numbers, focusing on combining tens and ones separately.

    Assessment: Student accurately solves 4/5 two-digit addition problems (no regrouping) using drawings or blocks.
